# Ravelry API documentation (offline copy)

`RavelryApiDocumentation.html` is a saved copy of the official Ravelry API reference
(https://www.ravelry.com/api), which is only viewable with a logged-in Ravelry account.
It is checked in so the endpoint reference is available offline and greppable when
working on `RavelryApiClient`.

Captured: July 2026. If Ravelry changes the API, re-save the page from a logged-in
browser session and replace this file (rewrite the `..._files/` asset references to
`assets/`).
